> ブログ制作ワークスペース：素材 → 気づき → ブログ構成 を一気通貫で

## 🎯 このツールが解決すること

外出先（本・映画・旅・セミナー）で得た気づきが、頭の中で消えてしまう問題を解決する。

```
モバイルで音声入力
    ↓
PCダッシュボードに自動表示
    ↓
AIと深掘り対話（グリル）
    ↓
マインドマップで見える化
    ↓
日英バイリンガル・ブログ構成を生成
```

---

## 📱 機能1：モバイル入力画面（/mobile または /input）

### 入力フィールド
| フィールド | 入力方法 | 選択肢 |
|---|---|---|
| カテゴリー | ドロップダウン | 本 / 映画 / 旅 / セミナー / その他 |
| 概要・内容 | 音声入力 or テキスト | 自由記述 |
| 気づき | 音声入力 or テキスト | 自由記述 |
| アクション | 音声入力 or テキスト | 自由記述 |

### 音声入力の仕組み
- ブラウザ内蔵のWeb Speech API（マイクボタンを押して話すだけ）
- 日本語・英語どちらでも認識
- 話し終わると自動でテキストに変換

## 🖥️ 機能2：PCダッシュボード（/dashboard）

### 4ペイン構成

```
┌─────────────────┬──────────────────────────────────────┐
│                 │                                      │
│  ① 素材リスト   │  ② AIグリル（深掘り対話）            │
│                 │                                      │
│  カテゴリー別   │  選択した素材について                 │
│  フィルター付き │  Claudeと対話する                    │
│                 │                                      │
├─────────────────┼──────────────────────────────────────┤
│                 │                                      │
│  ③ マインドマップ│  ④ ブログ構成                        │
│                 │                                      │
│  対話から       │  日本語記事の骨格                    │
│  自動生成       │  ＋英語サマリー                      │
│                 │                                      │
└─────────────────┴──────────────────────────────────────┘
```

### 各ペインの詳細

**① 素材リスト**
- モバイルから送られた素材が並ぶ
- カテゴリーでフィルター
- クリックすると②③④が連動

**② AIグリル**
- 選択した素材をベースにClaudeが質問を投げかける
- 「Take The Helm」読者向けのトーンに設定済み
- 対話履歴はセッション中保持

**③ マインドマップ**
- AIグリルの対話から自動でキーワード・つながりを抽出
- ビジュアルで思考を整理
- SVGで描画（ライブラリ：react-flow または d3）

**④ ブログ構成**
- 素材＋グリル内容から記事の骨格を自動生成
- 出力形式：タイトル候補 / 導入 / H2×3〜4 / まとめ / CTA
- 日本語メイン ＋ 英語サマリー（バイリンガル）
- Squarespaceにコピペしやすい形式

## 🗄️ データベース設計（Neon）

### 新規追加テーブル：`blog_materials`（素材）

```sql
CREATE TABLE blog_materials (
  id          SERIAL PRIMARY KEY,
  category    TEXT NOT NULL,        -- 本/映画/旅/セミナー/その他
  title       TEXT,                 -- 素材のタイトル（本名・映画名など）
  summary     TEXT,                 -- 概要・内容
  insight     TEXT,                 -- 気づき
  action      TEXT,                 -- アクション
  created_at  TIMESTAMP DEFAULT NOW(),
  updated_at  TIMESTAMP DEFAULT NOW()
);
```

### 新規追加テーブル：`blog_drafts`（ブログ構成）

```sql
CREATE TABLE blog_drafts (
  id              SERIAL PRIMARY KEY,
  material_id     INTEGER REFERENCES blog_materials(id),
  title_ja        TEXT,   -- 日本語タイトル候補
  title_en        TEXT,   -- 英語タイトル候補
  structure_ja    TEXT,   -- 日本語構成（JSON）
  summary_en      TEXT,   -- 英語サマリー
  status          TEXT DEFAULT 'draft',  -- draft/published
  created_at      TIMESTAMP DEFAULT NOW()
);
```
